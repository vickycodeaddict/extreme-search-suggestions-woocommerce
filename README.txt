Extreme Search Suggestion for WooCommerce plugin is very fast and powerful plugin to serve product on demand.

== Description ==

Extreme Search is a Fast and live search suggestion addon for WooCommerce.

It provides the best user experience and help to increase sales directly.

###Features

- Fast response even with thousands of products.
- Json based data serve instead of a database query.
- Doesn't effect on server as suggestions don't load database or core wordpress.
- Customizable Search and Display field.

###Options & Settings

This plugin provide many option for *Search* function and *Display* function. All option is given below


#### - Popup on search box:

It will show suggestion in full screen (Preferred)

#### - Regenerate on Edit Product:

It will Regenerate whole json or refresh all data on edit product. Leave it disable if you have huge product on your store.

#### - Suggestion Display Fields:

Selected fields will show in suggestion block of product.

#### - Suggestion Display Attributes:

Selected product attribute will show in suggestion block of product.

#### - Search Fields:

Selected field will enable search product by those field value.

#### - Search Attributes:

Selected attribute will enable search product by those attribute value.

#### - Exclude Product:

Add product IDs comma separated which you wonâ€™t display on search suggestion.

#### - Minimum keyword for suggestion:

Suggestion will start after minimum selected keyword on search field. (Preferred 2-3)

#### - Redirect on New Tab:

It will open new tab once user select product from suggestion.

###Troubleshoot

If your theme conflict with design or toggle searchbox then raise an issue in support forum.
Alternatively you can ask your developer to make it compatible, It will hardly take an hour.


== Installation ==

1. Download plugin zip file and extract it.
1. Upload `extreme-search` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Change Selector in plugin setting if required.

== Frequently Asked Questions ==

= Extreme Search plugin is compatable with any theme? =

Yes, Some theme has different search box format, In that case you can change selector from plugin setting page.

= Why Extreme Search plugin is so fast? =

Extreme Search plugin use json based data which serve instant result to user.

= Does this plugin effect on server load? =

No, Extreme Search plugin use json based data so it doesn't use database when user search for product.
